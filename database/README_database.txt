
------------------------------【数据库配置】------------------------------

修改config.php配置不同数据库连接参数，命令行使用配置中的配置id连接数据库

示例：

$db_argv = array(
    'db1' => array(
        'host' => '127.0.0.1',
        'user' => 'root',
        'pass' => 'wlwlwl',
        'name' => 'farm',
        'port' => 3306
    ),
    'db2' => array(
        'host' => '192.168.2.10',
        'user' => 'root',
        'pass' => 'wlwlwl',
        'name' => 'farm',
        'port' => 3307
    )
);


------------------------------【新建|更新数据库】--------------------------

示例：

    +-------------------------------------+
    | php main.php update|change db1      |
    +-------------------------------------+

执行changes目录下的文件
将 db1 所引用的数据库更新到changes目录中可以找到的最新版本

使用前建议先更新svn


------------------------------【导出数据库】------------------------------

示例：

    +-------------------------------------+
    | php main.php export db1             |
    +-------------------------------------+

将 db1 所引用的数据库中模版数据导出成 changes 目录中的.dump.php文件

文件名为当前数据库版本号 +1 ，导出完毕建议立即svn提交文件，避免版本冲突

此操作只用于模版数据库的导出，不要用于导出开发用数据库


------------------------------【备份数据库】------------------------------

示例：

    +-------------------------------------+
    | php main.php backup db1             |
    +-------------------------------------+

将 db1 所引用的数据库导出成 backup 目录下的.php文件，包含所有表


------------------------------【还原数据库】------------------------------

示例：
    +-------------------------------------+
    | php main.php restore db1 bk_*[.php] |
    +-------------------------------------+

将 backup 目录下的备份数据还原到 db1 所引用的数据库


------------------------------【修复数据库】------------------------------

示例：
    +-------------------------------------+
    | php main.php repair db1 fix_*[.php] |
    +-------------------------------------+

将 backup 目录下的备份数据还原到 db1 所引用的数据库
